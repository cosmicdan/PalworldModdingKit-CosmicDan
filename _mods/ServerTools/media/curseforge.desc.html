ğŸ§° Installation


Since this mod is for dedicated servers only, some intermediate knowledge is assumed:

  • Install UE4SS 3.0 or later: Download the release and extract to
    PalServer\Pal\Binaries\Win64\
     and start the server once to generate the mod folders;
  • Download the Server Tools mod from CurseForge and extract. Move the contents of each folder to the correct location:
    LogicMods -> PalServer\Pal\Content\Paks\LogicMods\
    Mods -> PalServer\Pal\Binaries\Win64\Mods\
     You should end up with the files:
    PalServer\Pal\Content\Paks\LogicMods\ServerTools.pak
    PalServer\Pal\Binaries\Win64\Mods\ServerToolsLoader\Enabled.txt
    PalServer\Pal\Binaries\Win64\Mods\ServerToolsLoader\Scripts\     [with one or more .lua files and foldes]
    ...start the server once to verify it worked and to generate default configuration files. Running the -Cmd server version is recommended for the console output.
  • The console/log should show some info along with the paths of the newly-generated config files. For example:
    Setting breakpad minidump AppID = 1623730
    [ServerTools] Server starting...
    [ServerTools] Welcome to ServerTools v0.1.0
    [ServerTools] New config file generated: D:/SteamGames/PalServer/Pal/Saved/ServerTools.config.json
    [ServerTools] Loaded config OK: D:/SteamGames/PalServer/Pal/Saved/ServerTools.config.json
    [ServerTools] New whitelist file generated: D:/SteamGames/PalServer/Pal/Saved/ServerTools.whitelist.json
    [ServerTools] Whitelist JSON seems OK, total IDs: 2
    [ServerTools] Server up!
  • You can edit the ServerTools.whitelist.json without restarting the server; it will be updated automatically when the next player joins. So add one or more steamId64's, for example:
    {"whitelistSteamIds":[
    "76561197982956148",
    "1"
    ]}
    ...in this example the first ID is me (CosmicDan) and the second is just an invalid example to show config syntax, invalid Steam ID's will simply be ignored;
  • Changes to ServerTools.config.json DO require a server restart to reload the config. The default ServerTools.config.json as of the latest version is:
    {
    "whitelistEnabled":true,
    "broadcastPlayerJoinLeave":true,
    "welcomeMsgEnabled":true,
    "welcomeMsg":"Welcome to my Palworld server!",
    "chatLogEnabled":true
    }
JSON is not ideal, I will be looking into a different markup format later.
